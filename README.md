# Portafolio Digital — Erick

Portafolio web personal desarrollado con **HTML** y **CSS**, con un diseño profesional, limpio y totalmente responsivo.

## Descripción

Este sitio presenta información personal, los servicios que ofrezco y las diferentes formas de contactarme. Está construido con una estructura sencilla de dos archivos para facilitar su mantenimiento y personalización.

## Estructura del proyecto

```
Portafolio-Digital/
├── index.html      # Estructura del sitio
├── styles.css      # Estilos y diseño visual
├── foto.jpg        # Foto personal (agregar manualmente)
└── README.md       # Este archivo
```

## Secciones

- **Inicio:** Presentación principal con llamados a la acción.
- **Acerca de mí:** Breve biografía junto a mi foto y habilidades.
- **Proyectos:** Lo que ofrezco como profesional.
- **Contacto:** Email, WhatsApp e Instagram con enlaces directos.

## Cómo usarlo

1. Descarga o clona los archivos en una carpeta.
2. Guarda tu foto personal como **`foto.jpg`** dentro de la carpeta del proyecto.
3. Abre **`index.html`** en cualquier navegador moderno.
