# Portafolio Digital — Erick

Portafolio web personal desarrollado con **HTML** y **CSS**, con una paleta cálida (crema + marrón + acento dorado), diseño moderno y totalmente responsivo.

## Descripción

Este sitio presenta información personal, los servicios que ofrezco, mis proyectos y diferentes formas de contacto. Está construido con una estructura sencilla basada en dos archivos principales para facilitar su mantenimiento y personalización.

## Estructura del proyecto

```
Portafolio-Digital/
├── index.html                         # Estructura del sitio
├── styles.css                         # Estilos y diseño visual
├── README.md                          # Este archivo
├── imagenes/
│   └── DSC01850 copia.jpg.jpeg        # Foto personal
└── Numero magico/
    └── Numero magico/
        └── NumeroMagico.html          # Proyecto enlazado
```

## Secciones

- **Inicio:** Presentación principal con llamados a la acción.
- **Acerca de mí:** Breve biografía junto a mi foto y habilidades.
- **Lo que ofrezco:** Servicios profesionales (Desarrollo Web, Apps Móviles, Software a Medida, Bases de Datos).
- **Proyectos:** Tarjetas de proyectos. La tarjeta de **Número Mágico** abre el proyecto en una nueva pestaña.
- **Contacto:** Email, WhatsApp e Instagram con enlaces directos.

## Cómo usarlo

1. Descarga o clona la carpeta del proyecto.
2. Abre **`index.html`** en cualquier navegador moderno.
3. Al hacer clic en la tarjeta **Número Mágico**, se abrirá el proyecto en una nueva pestaña.
